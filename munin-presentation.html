Infrastruktur & beroenden

Projektet fick ett rejält lyft i grunden — React uppgraderades till 18.3, @mui/styles fasades ut till förmån för @mui/material/styles, och ESLint konfigurerades om till airbnb v19. Alla gamla console.log-rader och oanvända komponenter städades bort.

Säkerhet & autentisering (Yggdrasil)

API:et gick från öppet till fullt skyddat. Alla GraphQL-resolvers skyddas nu med isAuthenticated och raderingsoperationer kräver dessutom isAdmin — båda via graphql-resolvers och ett JWT-flöde med SECRET ur miljövariabel.

  • Inloggning returnerar en JWT-token som klienten sparar i localStorage och skickar som Authorization-header vid varje request
  • API:et validerar token i kontexten — saknas den kastas FORBIDDEN direkt
  • Admin-bootstrapping: signUp-mutationen accepterar ett adminSecret för att kunna sätta admin-flaggan vid första uppstart
  • GraphQL-introspection inaktiverat i produktion — schemat exponeras inte utåt
  • Inloggningsfel ger ett generiskt meddelande — ingen information om huruvida användaren finns (förhindrar username enumeration)
  • Applikationen vägrar starta om SECRET-variabeln saknas (fail fast)

På klientsidan: utloggning rensar både JWT-token och Apollo-cachen (client.clearStore()) så att ingen data läcker mellan sessioner.

Tre språk — fullt genomfört

Hela UI:t är översatt till italienska, svenska och engelska med react-i18next. Alla texter — navigering, formulärfält, bekräftelsedialoger, felmeddelanden, temaknappar — lever i lokaliseringsfiler. I headern väljer man språk via landsflaggor och valet sparas i localStorage.

Bildhantering

Porträtt och bokomslag fick en genomtänkt hantering — bilderna lagras som filnamn i databasen och URL:en byggs från en miljövariabel. Standardbilder visas när foto saknas. Porträtten i listorna renderas som kvadratiska thumbnails med hover-preview.

Designsystem & teman

Det stora projektet i slutfasen. Vi gick från en hårdkodad mörkgrön färg till ett dynamiskt temasystem med tre teman:

🟤 Earth — varm beige 🔵 Ocean — stålblå 🌙 Dark — lila accent

Temavalet sparas i localStorage, data-theme sätts på body, och alla färger definieras centralt som CSS custom properties i _theme-vars.scss — inga hårdkodade färger i komponenterna.

Navigation & UX

  • Temaväljarens tre ikonerbaserade knappar (berg, vågor, måne) med MUI Tooltip som visas direkt vid hover
  • Utloggningsknapp som rensar JWT-token och Apollo-cache
  • DataGrid-tabellernas header och footer följer aktivt tema
  • Dynamisk listhöjd som anpassar sig till antal filtrerade rader

Mobilanpassning

Första steget mot mobile-first: deprecated media queries ersatta, navigation wrappas och komprimeras på liten skärm, sidmarginaler anpassas.

Det stora lyftet: på skärmar under 600 px ersätts DataGrid-tabellerna med en kortbaserad mobilvy för både författare och böcker — samma data, en layout byggd för touch. Korten följer aktivt tema via CSS custom properties.
