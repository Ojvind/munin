# Munin

Munin är en app för att hålla koll på författare och böcker du läst.

Uppkallad efter Odens korp som representerar minnet — varje dag flyger Munin (tillsammans med Hugin) ut
över världen och samlar berättelser åt Oden.

## Tech
- React
- Apollo Client
- GraphQL

## API
Munin kommunicerar med [Yggdrasil](https://github.com/Ojvind/yggdrasil) —
ett GraphQL API byggt med Apollo Server, Node.js och MongoDB.

## Fonts
There are several different ways to implement and use fontawesome icons. I chosed to install the free packages and import the icons I wanna use in each file.
See "Using Icons via Individual Use" here:
https://fontawesome.com/how-to-use/on-the-web/using-with/react

## Installation

* `git clone git@github.com:Ojvind/brage.git`
* cd brage
* yarn install
* yarn start
* visit `http://localhost:3000`
